
                    PORTFOLIO WEBSITE
                 Developed by: Mohit Tanwar



1. PROJECT INTRODUCTION

This is a simple, responsive Portfolio Website that is built 
with HTML and CSS. It is composed of Home, About, Projects, 
Skills, and Contact sections. 


The site presents personal details, technical skills, projects, 
and a functioning framework for a contact form.




2. KEY FEATURES

• Responsive and clean design  
• Sticky navigation menu  
• Hero section with introduction  
• An About section with a profile image  
• Projects with description  
• Skills table with style  
• Contact form  
• Back-to-top button  
• Google Fonts (Poppins)  
• Good accessibility (Skip-to-content link)




3. FILE STRUCTURE

project-folder/
│
├── index.html   → Main HTML page  
├── style.css    → Stylesheet  
└── My Profile Picture.jpeg




4. HOW TO RUN THE PROJECT

1. Download all files for the project. 
2. Ensure "style.css" is in the same folder as 
  "index.html". 
3. Double-click "index.html" to open in a web browser. 
4. The site will load in the browser.




5. TECHNOLOGIES

• CSS3  
• Google Fonts (Poppins)
